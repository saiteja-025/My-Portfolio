# 🚀 Sai Teja Portfolio

A premium, production-ready, and highly interactive software engineering portfolio. Built with modern web technologies, this site features responsive grids, fluid animations, and custom layouts tailored to show expertise in **Software Development** and **Machine Learning**.

---

## ⚡ Tech Stack & Architecture

- **Core Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS v3 (JIT Compilation, dark-mode support)
- **Animations:** Framer Motion (Smooth scroll reveals, floating ambient glows, keyframe effects)
- **Icons & Branding:** Lucide React, FontAwesome, and Simple Icons (via `react-icons`)
- **Routing:** React Router v7 (Single Page Application configuration)
- **Forms & Redirection:** FormSubmit serverless endpoint integration

---

## ✨ Features

- **Split Hero Section:** Displays a professional suit portrait alongside large, high-impact name typography. Features pulsing background glows and rotating gradient borders that respond to user hover.
- **Skills Category Grid:** Modular presentation of competencies grouped by languages, frontend, backend, AI/ML, and tools.
- **Coding Profiles:** Dedicated section showcasing DSA and competitive programming presence on LeetCode with glowing orange hover cards.
- **Project Showcase:** Interactive project showcase grid routing to individual project case study subpages.
- **Contact Socials:** Direct integration of primary professional channels—GitHub, LinkedIn, LeetCode, and Email.
- **Responsive Architecture:** Flex-based ordering and grid structures that stack components beautifully on mobile and split them wide on desktop.

## 🛠️ Installation & Local Development

To clone, set up, and run this repository locally:

### 1. Clone the repository
```bash
git clone https://github.com/saiteja-025/My-Portfolio.git
cd My-Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start development server
```bash
npm run dev
```
Open **[http://localhost:5173](http://localhost:5173)** in your browser to view the site.

### 4. Create a production build
```bash
npm run build
```
The compiled, optimized bundle is written to the `./dist/` directory.
